# Project Rules & Guidelines

- I define my goal clearly before asking AI
- I break my project into small parts
- I write detailed prompts (tech stack + requirements)
- I review and test all AI-generated code
- I never share sensitive data
- I use official documentation
- I improve step by step, not all at once

## Coding Standards
- Write clean and optimized code
- Use best practices
- Add comments to explain code
- Focus on security
- Do not assume missing details, ask instead
- Act as a senior full-stack developer
- Clean code

- Scalable architecture
